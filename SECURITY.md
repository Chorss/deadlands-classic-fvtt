# Security Policy

## Supported Versions

Only the latest release of Deadlands Classic — Community Edition receives security updates.

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

This project is a Foundry VTT game system — it runs entirely client-side within a trusted game session and handles no sensitive personal data, authentication, or network services beyond standard Foundry VTT.

If you discover a security issue (e.g., XSS via user-supplied data rendered in templates, or unsafe eval usage), please **do not open a public GitHub issue**.

Instead, report it privately by emailing the maintainer or using [GitHub's private vulnerability reporting](https://github.com/Chorss/deadlands-classic-fvtt/security/advisories/new).

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an acknowledgement within 7 days and a fix or mitigation within 30 days for confirmed issues.
